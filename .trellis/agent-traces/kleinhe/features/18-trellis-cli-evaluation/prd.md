## 可能的 Trellis CLI 形态

如果决定做，可能的形态：

### 形态 A：轻量 CLI Wrapper

```bash
trellis start          # 启动 Claude Code + 注入 context
trellis session end    # 结束并保存对话
trellis history list   # 查看历史
```

- 本质上是 Claude Code 的增强层
- 最小开发量

### 形态 B：独立 CLI

```bash
trellis chat           # 自己的对话界面
trellis code           # 代码编辑模式
trellis review         # 代码审查
```

- 完全独立的 AI 编程助手
- 开发量大，需要考虑 ROI

### 形态 C：IDE 插件

- VS Code / JetBrains 插件
- 与现有 IDE 深度集成
- 中等开发量
